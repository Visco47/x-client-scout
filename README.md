# X Client Finder

Discover potential clients on X (formerly Twitter) based on your skills and business ideas.

## Theme

Dark Glassmorphic Design
- Background: Deep Navy Blue (#1A2238)
- Accent: Electric Blue (#3498DB)
- Glassmorphic Overlay: Transparent White (#FFFFFF1A)

## Features

- Input X handle and skills
- AI-powered client discovery
- Pagination of client results
- Futuristic dark theme

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
